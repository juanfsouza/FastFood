### 🚀 **Instalação e Uso**
1. **Clone este repositório**

 ```bash
 git clone https://github.com/juanfsouza/FastFood.git
 ```

### Navegue até a pasta do projeto
```bash
cd fastfood
```
### Instale as dependências
```bash
npm install
```
### Execute o projeto
```bash
npm run dev
```
